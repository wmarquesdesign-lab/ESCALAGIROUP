# GIROUP Escala

Projeto pronto para publicar no Vercel, Netlify ou GitHub Pages.

Arquivos principais:
- index.html
- manifest.json
- service-worker.js
- giroup-logo.png

No Vercel:
1. Envie estes arquivos soltos para o GitHub, sem ZIP dentro do repositório.
2. Importe o repositório no Vercel.
3. Framework Preset: Other.
4. Build Command: deixe vazio.
5. Output Directory: deixe vazio.
6. Clique em Deploy.
