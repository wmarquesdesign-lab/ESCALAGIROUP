GIROUP - SOLUÇÕES PARA VAREJO

Como publicar na Netlify:
1. Extraia este ZIP.
2. Entre na Netlify > Add new site > Deploy manually.
3. Arraste a pasta extraída inteira para a tela da Netlify.

Senha padrão para entrar: wagner1818
Você pode trocar dentro do app em Configurações.

Módulos incluídos:
- Dashboard GiroUp
- Estoque completo importado
- Busca por código e nome
- Histórico de custo por produto
- Comparação de compras/entradas
- Importação de CSV
- Exportação CSV
- Atalhos para Giroup Cartaz, Criador de Prompt e Escala de Folga
